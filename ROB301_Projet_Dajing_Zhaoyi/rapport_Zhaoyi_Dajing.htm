Landing Gear System
ROB301 Projet

Summary:This is the final report of the project of the course ROB301, which realizes the software part of a landing gear system. The total system is described in the operator "Total_System"
Some elements to be specified in addition to the file "landing_system.pdf":
The unity of all of the durations is 100ms
The handle is "true" means the handle is in the position "up". The handle is "false" means the handle is in the position "down".
The analogical switch is open when it is "true", the analogical switch is close when it is "false".
In the operator "Computing_Module", we assume that the handle cannot be changed within 100ms after each operation.
